## ✨ 功能特点

### 📱 智能 Widgets

- **时钟** - 实时显示时间和日期，支持 12/24 小时制切换
- **天气** - 实时天气信息，支持地理定位和多城市管理
- **搜索** - 多搜索引擎快速切换（Google、Bing、百度、DuckDuckGo）
- **待办事项** - 简洁的任务管理，支持拖拽排序和完成标记
- **可扩展** - 支持自定义 Widget 开发

### 🎨 高度可定制

- **主题切换** - 深色/浅色模式一键切换
- **背景选择** - 支持纯色、渐变、图片、随机壁纸
- **灵活布局** - 网格系统支持自由拖拽调整位置和大小
- **快捷方式** - 自定义网站快捷方式，支持拖拽排序
- **统一样式** - 所有 Widget 遵循一致的视觉规范

### 🚀 性能优势

- **按需加载** - Widget 按需动态加载，提升初始加载速度
- **离线可用** - PWA 支持，离线也能正常使用
- **数据持久** - 所有配置自动保存到本地
- **响应式** - 完美适配各种屏幕尺寸

### 🎯 用户体验

- **拖拽操作** - 直观的拖拽交互，轻松调整布局
- **即时预览** - 设置更改即时生效
- **深色模式** - 全面支持深色模式，保护您的眼睛
- **数据导入导出** - 轻松备份和迁移您的配置

## 🎨 自定义配置

### 添加快捷方式

1. 点击右下角的"编辑"按钮
2. 点击"+"号添加新快捷方式
3. 输入名称和 URL
4. 拖拽调整位置

### 配置 Widgets

1. 点击右下角的"设置"按钮
2. 切换到"组件"标签
3. 启用/禁用需要的 Widget
4. 在"外观"标签调整背景和主题

### 更改背景

1. 打开设置面板
2. 选择背景类型（颜色/渐变/图片/随机）
3. 调整模糊、亮度、透明度效果

## 🌟 核心特性

### Widget 系统

- **动态加载** - 仅加载启用的 Widget，节省资源
- **错误隔离** - 单个 Widget 故障不影响整体运行
- **可扩展** - 支持自定义 Widget 开发

### 数据管理

- **自动保存** - 所有更改自动持久化到本地
- **导入导出** - 支持配置文件的导入导出
- **隐私保护** - 所有数据存储在本地，不上传服务器

### 响应式布局

- **网格系统** - 基于 96x96px 单元格的精确布局
- **自适应** - Widget 根据容器大小自动调整显示
- **拖拽调整** - 拖动边缘即可调整 Widget 大小

## 📝 常见问题

### 如何重置所有设置？

在浏览器开发者工具中清除 `localStorage` 中的 `echo-tab-storage` 键，然后刷新页面。

### 天气信息不显示？

1. 确保已启用天气 Widget
2. 允许浏览器获取地理位置
3. 或手动在设置中搜索城市

### 数据会丢失吗？

所有数据保存在浏览器本地存储中，清除浏览器缓存会导致数据丢失。建议定期使用"导出配置"功能备份数据。

### 支持哪些浏览器？

现代浏览器均支持，推荐使用 Chrome、Edge、Firefox、Safari 最新版本。
